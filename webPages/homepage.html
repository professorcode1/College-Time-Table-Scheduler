Here's how you use the generator. First fill all the appropriate information via vising the links below.
Professor,groups and rooms are resources and the collections of these togther are called a period.
Each resourse can be given ban-times (time when the resource cannot be allocated, for example you have 5 days and 8 periods, and a prof need to go home early on thursday,you can set (4,8) as anti-time)
A period can also haev a set-time, if a period need to take place at a certain time on a certain day that can be set as well. First set the days and period of the college.
Then add all the professors rooms and groups in your institute.
Then create a coruse, which is taught by a set of professors, to a set of groups.
Then for every course,generate periods.Once you have made all the period,click on generate time table.
